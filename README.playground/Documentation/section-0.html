MarkdownToPlayground

Aka mkdtoplg

This is a simple tool that will convert your markdown documentation to a playgorund documentation format for the Swift language. It is super easy to use you just create your markdown file or take it from a github README file like this one and feed it to tool mkdtoplg file.md. The tool will parse all the markdown to a well formatted Playground file, dividing all the HTML from the swift code.
